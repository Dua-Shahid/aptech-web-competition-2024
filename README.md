Aptech Web Competition 2024

This project was built for the Aptech Web Design and Development Competition 2024 using HTML, CSS, and JavaScript.

It is a custom-designed website created from scratch without using any frameworks. The main purpose of this project was to improve front-end development skills, UI design, and creativity.
